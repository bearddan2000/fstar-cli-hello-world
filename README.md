# fstar-cli-hello-world

## Description
A POC for fstar programming language.

## Tech stack
- ocaml
    - opam
- python

## Docker stack
- ubuntu:22.04
- rzuckerm/fstar

## Build note
This took 10+ minutes to build.

`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
